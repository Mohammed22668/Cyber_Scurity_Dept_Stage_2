# Assignment: System Info Collector (Cybersecurity Practical)

## Question:

Create a Python class named `SystemInfo` that collects basic information about the user’s computer/network. The class should include useful methods that are relevant for cybersecurity tasks.

## Class Requirements

The class should include the following methods:

- `get_local_ip()` – returns the computer’s local IP address.
- `get_public_ip()` – returns the public IP address of the system.
- `get_mac_address()` – returns the MAC address of the main network interface.
- `get_wifi_password()` – returns the saved Wi-Fi password.

## Task

1. Create one object of the class.
2. Call all methods and print the results.
3. _(Optional)_ Make the code show the result as GUI
